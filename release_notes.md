# Akili v1.0.0

Initial production release.

## Features
- **AI Curriculum Generator** — Personalised course outlines with module-by-module structure tailored to your education level and country.
- **Smart Lessons** — AI generates detailed lessons for each module with clear learning objectives and step-by-step explanations.
- **Interactive Quizzes** — Auto-generated multiple-choice questions with instant feedback. Always free.
- **Mock Exams** — Timed full-length assessments with automatic grading and performance analytics.
- **AI Tutor** — Chat with Akili, an AI tutor that answers questions with web-sourced, cited explanations.
- **Gamification** — XP, levels (Freshman → Genius), daily streaks, and achievement badges.
- **Credit System** — 150 daily credits for AI actions. Quizzes are always free.
- **Dark/Light Mode** — System-aware theme switching.
- **Offline-First** — SQLite with WAL mode. Lessons cached locally after first generation.
